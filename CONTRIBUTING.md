# Contributing — MPL-D

**Maturity:** Process stub. No open contribution workflow is active.

## Scope

This repository documents the Counter-UAS Multi-Point Laser Dazzler Prototype
(MPL-D internal codename). Changes must preserve intellectual honesty: maturity
labels, explicit limitations, and conservative estimates.

## Before you edit

1. Read [`README.md`](README.md) and [`docs/phase0_gate_status.md`](docs/phase0_gate_status.md).
2. Do not add performance guarantees, operational range claims, or unlabeled test results.
3. Do not commit secrets (`.env`, credentials, export-controlled attachments).

## Pull requests

Internal Fratres X AI contributors only unless program leadership authorizes
external access. PRs should state which maturity gate or document section is
affected and whether bench/safety claims changed.

## Questions

Contact Fratres X AI project leadership. Do not open public issues containing
export-controlled or operational details.
